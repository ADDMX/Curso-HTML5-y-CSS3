Imagenes Lossy y Lossless

Existen dos tipos de imágenes, sin pérdida o con pérdida, y esto depende de cómo el formato maneja las imágenes.

Formatos de imágenes Lossless

Estos formatos de imagen no pierden nada de calidad y conservan su estado original sin perdida de peso para la web

Formatos de imágenes Lossy

Estos formatos de imagen pierden un poco su calidad pero ayudan a reducir el peso en los navegadores web

Tipos de formatos de imagenes para web

  • GIF (gif animados): este tipo de formato es sin perdida de peso para la web
  • PNG-8: es una imagen formado por 256 colores y que conservan transparencia se utilizan en logotipos.
  • PNG-24: es una imagen que tiene mas de 256 colores y es un formato mas pesado para web
  • JPG/JPEG: es un tipo de imagen Lossy hay perdida pero este formato es para fotografia maneja una gama de colores ilimitada.
  • SVG-Vector/Lossless: es un tipo de formato vector muy ligero para iconos o logotipos para escalar y esta echo para pantalla de retinas y no se pierde calidad en la imagen.

Optimización de Imagenes
